🎨 Cooder Designer Eğitmeni

Cooder (👨🏾‍💻) ile modern, responsive, dark-mode HTML/CSS/JS arayüzleri tasarla

📚 Başlangıç

Cooder'ın Designer rolü, kullanıcının HTML/CSS/JS ihtiyaçlarını karşılar. Basit yazımla kompleks arayüzler üretebilir.

Kullanım

/designer bir dashboard oluştur: koyu tema, kartlar, grafikler

Cooder, bu talebi alır ve cooder_agent.py içindeki designer() fonksiyonu ile işler.

🧩 Component Kütüphanesi

Cooder, aşağıdaki component'leri otomatik üretebilir:

🔘 Butonlar

<button class="btn btn-primary">Ana Buton</button>
<button class="btn btn-secondary">İkincil</button>
<button class="btn btn-danger">Sil</button>

📋 Kartlar (Card)

<div class="card">
  <h3>Başlık</h3>
  <p>İçerik...</p>
</div>

📊 Tablolar

<table class="table">
  <thead><tr><th>ID</th><th>İsim</th></tr></thead>
  <tbody><tr><td>1</td><td>Test</td></tr></tbody>
</table>

📈 Formlar

<form class="form">
  <input type="text" placeholder="Adınız">
  <button type="submit">Gönder</button>
</form>

🌙 Dark Mode Tema Sistemi

Tüm component'ler CSS variables ile dark-mode uyumludur:

:root {
  --bg: #0f172a;
  --card: #1e293b;
  --text: #e2e8f0;
  --primary: #3b82f6;
  --success: #10b981;
  --warning: #f59e0b;
  --danger: #ef4444;
}
body { background: var(--bg); color: var(--text); }
.card { background: var(--card); border: 1px solid var(--border); }

Bu, tüm raporlarınızda (Meta Glasses, Flight Search, Security) kullanılan aynı tema!

📱 Responsive Tasarım

Cooder otomatik olarak mobile-first tasarımlar üretir. Grid/Flexbox kullanımı:

.container { max-width: 1200px; margin: auto; padding: 20px; }
.grid { display: grid; grid-template-columns: repeat(auto-fit, minmax(300px, 1fr)); gap: 20px; }
@media (max-width: 768px) { .grid { grid-template-columns: 1fr; } }
